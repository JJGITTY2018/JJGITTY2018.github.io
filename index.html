Find Your Buddy

Description:

A hackathon front-end concept app built within 18-hours. The app utilizes RescueGroups.org API to pull available animal profiles for adoption. The passion and idea behind this project were to help organizations and users shorten the lengthy paperwork of adopting pets in shelters and to save an animal from being euthanized.

Role:

Assisted the team with UI and UX design. Designed the Home Page, Cat and Dog Page, and Single Animal Page. Worked on API authorization and pet display profiles.

Project Hurdle:

One of the challenges for this project was correctly using the API key in the HTTP Authorization header and retrieving data from the API. Our expectation was that API keys are delivered through the standard URL call. However, we soon learned that this feature was removed from this API version.

Solution:

I guided the team to continued with this API and not abandon it for a fake-generate pet database because of the short time frame we had. I provided other users example based on HTTP Authorization calls and carefully looked through the API documentation. With this help, our team was able to get the authorization to work and have the API response to us.
