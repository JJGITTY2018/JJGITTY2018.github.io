Description:

An original ProjectSienna(Beta) App created to replace an aging legacy flash app that was mounted on the walls of the Visitors Administration Building at the Queens Botanical Garden. It was later published in the Windows 8.1 Store to be used widely. Small parts of the original app were later migrated to PowerApps(Preview) and utilized to help internal staff with information processes such as incoming guests and membership information. The app was an initiative begun by my desires to learn how to create an application.

Role:

Developer - Designer

Project Challenges:

The initial challenges for this project were the original app ran on Macromedia Adobe Flash, it was outdated and no documentation or editing tools available for the app. The second challenge was the outdated hardware. Still running on Windows 2000 and XP and with touchscreens monitors with legacy IO ports the whole entire PCs needed to be upgraded. There was also a very limited amount of budget for this project.

Solution:

The app was built from scratch using only the texts and available images on existing websites. After careful planning and research, I found Project Siena(Beta) from the Microsoft team. The application was deployed locally with restried user controls functionality (so that visitors cannot access restrict of the PC). The hardware challenge was solved using budgeted mini-pc and newer touchscreen monitors were installed. 

Notable Things:

  • Scrolling Menu Navigation Bar
  • App Translation Selection
  • Colorful Photos
  • Used by Visitors and Teacher during visits and tours.
